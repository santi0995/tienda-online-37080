## Instalación

Aquí encontrarás una serie de instrucciones para instalar el proyecto:

1. Clonar el repositorio
   ```sh
   git clone https://github.com/santi0995/tienda-online-37080.git
   ```
2. Instalar paquetes NPM 
   ```sh
   npm install
   ```
   
3. Iniciar el proyecto
    ```sh
   npm start
   ```
